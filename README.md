# Flappy Bird Unity

A basic functional clone of the classic Flappy Bird game built using **Unity**.  
This version focuses on the core mechanics: jumping, obstacles, and score counting.

## 🎮 Features

- Tap `Space` to jump
- Moving pipes as obstacles
- Gravity and basic collision detection
- Score increases as you pass pipes
- Game resets on collision

## 📦 Download

Download the full project as a `.zip` file:  
**[flappybird.zip](./flappybird.zip)**

> Contains all project files. Open in Unity to run.

## 🚀 Getting Started

1. Unzip the file
2. Open the folder in Unity (tested with Unity 2020.3 or newer)
3. Open the main scene and press `Play`

## 🎯 Controls

- `Space`: Make the bird jump
- Avoid hitting the pipes or falling
